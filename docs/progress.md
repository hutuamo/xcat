# xcat 项目进展

## 2026-02-09: v0.2.0 Rust 重写完成
- 从 C++ 完整重写为 Rust
- 技术栈：pulldown-cmark + ratatui + crossterm + unicode-width
- 引入 FileFormat trait 实现格式可扩展架构
- 支持嵌套列表（C++ 版本不支持）
- 保留全部功能：标题、段落、代码块、列表、引用、水平线、表格、粗体/斜体/行内代码
- 终端渲染、HJKL 导航、状态栏均保持一致

## 2026-02-09: v0.1.0 初始版本完成
- 搭建项目结构，集成 md4c 和 ncurses
- 实现 Markdown 解析：标题、段落、列表、代码块、引用、分割线、加粗/斜体
- 实现终端渲染：颜色、样式属性、状态栏
- 实现浏览控制：HJKL导航、半页滚动、跳转、q退出
- 支持终端 resize
